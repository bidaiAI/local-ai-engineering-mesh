# Poster Spec

Use this file to turn the repository into a clean one-page poster or share image.

## Goal

Communicate three ideas in under 10 seconds:
1. this is not a prompt pack
2. this upgrades multiple AI tools under one operating law
3. it solves workflow collapse when a single tool becomes limited or inconvenient

## Main headline options

### English
- **Switch tools, not standards.**
- **Upgrade your AI workflow, not just one AI tool.**
- **One operating law across Codex, Claude, Cursor, and Antigravity.**

### 中文
- **工具可以切，标准不要重来。**
- **升级的是 AI workflow，不只是某一个 AI 工具。**
- **让 Codex、Claude、Cursor、Antigravity 共享一套 operating law。**

## Recommended layout

### Top strip
- headline
- short subheadline

### Center
- shared law node
- four tool boxes
- one converged project runtime box

### Bottom strip
- evidence gate
- release discipline
- reusable patterns
- short CTA with repo URL

## Recommended subheadline

### English
A public-safe operating model for turning local AI tools into one governed, memory-backed, evidence-driven engineering system.

### 中文
一套公开安全的 operating model，把本地 AI 工具组织成有治理、有记忆、有证据闭环的工程系统。

## Box order

1. Shared Law
2. Codex
3. Claude
4. Cursor
5. Antigravity
6. Project Runtime
7. Evidence Gate
8. Release Discipline
9. Reusable Patterns

## Caption / footer options

### English
- Quota can fail. A governed workflow should not.
- Start with one tool. Grow into a mesh later.
- Different tools, same quality bar.

### 中文
- 额度会断，workflow 不该断。
- 先从一个工具开始，之后再长成 mesh。
- 不同工具，同一质量门槛。
