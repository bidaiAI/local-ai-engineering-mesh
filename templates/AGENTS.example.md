# AGENTS.example.md

## Purpose
This file is a starter shared-law template for local AI engineering workflows.
Use it as the top-level operating law across one or more AI tools.

## Core rules
- Plan before broad implementation when uncertainty is high.
- Prefer the smallest sufficient change.
- Require evidence before calling work done.
- Treat publish, deploy, and external writes as gated actions.
- Promote repeated successful patterns into reusable workflows, templates, or skills.

## Working loop
- research
- plan
- execute
- verify
- publish carefully

## Evidence expectations
- tests when available
- logs when useful
- screenshots for visual changes
- review notes for risky diffs

## Memory expectations
- keep stable operating rules globally
- keep project conventions in project memory
- keep task discoveries session-local until they are durable
