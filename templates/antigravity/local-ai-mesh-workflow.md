# Local AI Mesh Workflow

Use this as a starting workflow for Antigravity.

## Purpose

Treat Antigravity as one endpoint inside a larger local AI engineering mesh.

## Workflow

1. read the shared operating law
2. identify whether the task belongs in Codex, Cursor, Antigravity, or a handoff sequence
3. prefer research-first and plan-first behavior on large tasks
4. return evidence, not only narrative
5. record durable knowledge when a pattern repeats
