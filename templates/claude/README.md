# Claude Templates

These files are public-safe starter assets for the Claude side of the local AI engineering mesh.

Use them when you want:
- a Claude-specific project law file
- a reusable commands pack
- a modular rules pack
- Claude to follow the same operating law as Codex, Cursor, and Antigravity

Suggested placement:
- project law: project root as `CLAUDE.md`
- commands: your Claude commands location
- rules pack: project or user-level Claude rules directory
