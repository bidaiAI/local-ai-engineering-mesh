# Codex Templates

These files are starter pieces for the Codex side of the local AI engineering mesh.

Use them when you want:
- a Codex-specific instructions layer
- a public-safe config stub
- Codex to plug into the same shared law as the other tools

Suggested placement:
- config: your Codex home config location
- instructions: your Codex instructions location
- project runtime: `<project>/.codex/`
